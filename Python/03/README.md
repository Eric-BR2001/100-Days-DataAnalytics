# Day 3: Python Function Exercises - Strings and Files

This day focuses on string manipulation and file I/O operations using Python.

## Topics Covered

1. Reversing a string
2. Counting vowels
3. Checking for palindromes
4. Word frequency analysis
5. Replacing words in text
6. Reading files
7. Writing to files
8. Appending to files
9. Counting lines in a file
10. Finding unique words

## Usage

Run the Python script to test all examples:

```bash
python day3_string_file_exercises.py
```
